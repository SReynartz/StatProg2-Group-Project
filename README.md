## Group Members

## Dataset Description
This repository contains an analysis of monthly traffic accident data from the Open Data Portal of the City of Munich. The dataset records monthly counts of traffic accidents categorized by type (e.g., alcohol-related accidents, hit-and-runs) and detail (e.g., total, injured) from 2000 to 2026.

## How to Run
All R scripts are designed to be executed from the project root directory.

1. **Clean Data**: `Rscript scripts/01_clean_data.R`
2. **Exploratory Analysis**: `Rscript scripts/02_exploratory.R`
3. **Predictive Modeling**: `Rscript scripts/03_predictive.R`
